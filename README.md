# Rogue Grid

A rogue-like puzzle game where you clear lines, earn upgrades, and survive against increasing difficulty.

## How to Play
- **Objective:** Clear lines to score points and survive as long as possible.
- **Controls:**
  - **Move:** Arrow Keys or WASD
  - **Rotate:** Up Arrow or W / Z
  - **Hard Drop:** Spacebar
  - **Hold:** C or Shift
- **Progression:**
  - Clearing lines fills your XP bar.
  - Level up to choose from 3 random upgrades.
  - Upgrades include passive bonuses, active consumables, and board modifications.

## Features
- **Roguelike Elements:** Random upgrades, increasing difficulty, and permadeath.
- **Physics System:** Gravity mechanics for floating blocks.
- **Dynamic Board:** The board expands and evolves as you play.
- **Consumables:** Use powerful abilities like Row Clear, Column Clear, and Gravity Shake.

## Development
Built with vanilla HTML, CSS, and JavaScript. No external dependencies required.
