# Rogue Grid

A rogue-like puzzle game where you clear lines, earn upgrades, and survive against increasing difficulty.

## How to Play
- **Objective:** Clear lines to score points and survive as long as possible.
- **Controls:**
  - **Move:** Arrow Keys or WASD
  - **Hard Drop:** Spacebar
  - **Hold:** Q
  - **Drop:** E
- **Progression:**
  - Clear missions to level up
  - Level up to choose from 3 random upgrades.
  - Upgrades include passive bonuses, active consumables, and board new pieces.

## Development
Built with vanilla HTML, CSS, and JavaScript. No external dependencies required.
